# Eclipse IDE Installation
1. Go to: https://www.eclipse.org/ide/
2. Select “Download”
3. Once the download is completed, run the file and select to install Eclipse for Java Developers
4. Select Java JDK 17.0.2 (LTS) on the Java 11+ VM section, select INSTALL
5. Select “Accept Now” on the User Agreement screen
6. Once the installation is complete, you can check out the “show readme file” but more importantly, select Launch
7. When it launches, select an appropriate location on your local machine for the eclipse-workspace folder
8. Click the “Use this as the default and do not ask again” checkbox and proceed to hit the “Launch” button
